## 📦 What is sts2-quickRestart?  

sts2-quickRestart is a simple mod for Slay the Spire 2 that lets you save and load your game quickly. Instead of going through multiple menus, you can jump back to an earlier point with a single action. This makes testing strategies and retrying runs faster and easier.  

The mod works on Windows computers and requires no technical skills to use.
